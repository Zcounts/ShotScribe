# Archived Documentation and Legacy Assets

This directory contains historical materials that are not part of the active build/deploy/runtime flow.

## Structure

- `migration/` — superseded migration-phase plans/checklists retained for history.
- `legacy-static/` — static HTML prototypes/legacy landing artifacts not used by current build.
- `assets/` — non-runtime reference assets moved out of repository root.

If a document here is needed operationally, link to it from an active runbook instead of moving it back into active paths.
